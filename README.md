  ## Setup
  How to start the program:-
  1. Clone the repo.
  2. Change the credentials in **NodeMCU.ino** file.
  3. Make sure you have `ESP8266WiFi.h` and `FirebaseArduino.h` in your arduino.
  4. Verfiy and Compile.
